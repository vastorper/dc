Como instalar o mas bien utilizar esta version en español

1. Primero de todo es recomendable q hagan una backup de sus cartas de METW completa en la carpeta graphics del gccg
2. Entren a la carpeta /ES q contiene el pack de cartas en español copien y peguen dentro de la carpeta original de cada expansion (ejemplo: Wizards) y reemplacen las existentes.
3. A disfrutar
